TIPOGRAFIA

	A tipografia desempenha um papel crucial na aparência e na legibilidade de um site. O Bootstrap 5 oferece uma série de classes e estilos para facilitar 
o controle sobre o texto e a tipografia em seu projeto. 

	Aqui estão alguns aspectos-chave relacionados à tipografia no Bootstrap 5:

	1. **Fontes e Estilos de Texto:**
   	O Bootstrap 5 inclui classes para definir tamanhos de texto, estilos (negrito, itálico, sublinhado) e alinhamentos. Você pode aplicar essas classes 
diretamente ao texto que deseja estilizar. 

	Por exemplo:

   		<p class="text-primary">Este é um parágrafo com cor primária.</p>
   		<p class="text-muted">Este é um parágrafo com cor desbotada.</p>
   		<p class="text-center">Este é um parágrafo centralizado.</p>
   		<p class="font-weight-bold">Este é um texto em negrito.</p>

	2. **Sistema de Tamanhos de Texto:**
  	O Bootstrap 5 introduziu um sistema de tamanhos de texto mais consistente e flexível, usando classes como `fs-1`, `fs-2`, ... `fs-6` para definir 
tamanhos de texto diferentes. Isso permite um controle mais preciso sobre o tamanho do texto em vários dispositivos.

   		<p class="fs-4">Este é um texto de tamanho 4.</p>

	3. **Cabeçalhos:**
   	Os cabeçalhos HTML (h1 a h6) também podem ser estilizados usando classes do Bootstrap para alcançar uma aparência consistente e agradável.

   		<h1 class="h1">Título Principal</h1>
   		<h2 class="h2">Título Secundário</h2>

	4. **Classe de Espaçamento de Linha:**
   	A classe `line-height` permite ajustar o espaçamento entre linhas para melhorar a legibilidade do texto.

   		<p class="line-height">Este é um parágrafo com espaçamento de linha ajustado.</p>

	5. **Classe de Espaçamento de Texto:**
   	A classe `letter-spacing` pode ser usada para controlar o espaçamento entre caracteres.

   		<p class="letter-spacing">Este é um parágrafo com espaçamento entre caracteres.</p>

	6. **Google Fonts:**
   	Enquanto o Bootstrap não integra diretamente Google Fonts, você pode facilmente adicionar fontes personalizadas ao seu projeto seguindo as práticas 
padrão do CSS.

	Lembre-se de que a tipografia não se limita apenas a esses exemplos. O Bootstrap 5 oferece uma ampla gama de opções de estilo de texto e tipografia 
para atender às suas necessidades de design. Sempre consulte a documentação oficial do Bootstrap 5 para obter informações mais detalhadas e atualizadas sobre
como trabalhar com tipografia.
